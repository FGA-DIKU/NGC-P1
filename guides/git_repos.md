To download git repos navigate to https://cld076-0006.cld076.vmc

1. Copy the GitHub HTTPS URL including ".git" into the Repository URL box, e.g. "https://github.com/Sllambias/asparagus.git"
2. Click "Add Mirror" - it should say "Repository updated successfully" in the top of the page. If it doesn't try to delete an entry from the list to make sure there's space to show your Repository.
3. Copy the URL from the "Local Repository"
4. Git clone the repository USING THE LOCAL URL in your terminal, e.g. "git clone https://cld076-0006.cld076.vmc/git/github.com/Sllambias/asparagus.git"

To update/pull changes repeat 1-2 and then do "git pull" inside the repository.
